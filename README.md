# 🪨📄✂️ Pedra Papel Tesoura Online

Jogo multiplayer online de Pedra, Papel e Tesoura desenvolvido com:

- HTML5
- CSS3
- JavaScript
- Node.js
- Express
- Socket.IO


# 🎮 Sobre o jogo

Dois jogadores entram em uma sala privada através de um código.

Cada jogador escolhe:

🪨 Pedra

📄 Papel

✂️ Tesoura


O servidor recebe as escolhas, calcula o vencedor e atualiza o placar em tempo real.
